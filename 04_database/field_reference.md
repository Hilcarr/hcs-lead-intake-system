# Field Reference

Main database fields used by the platform.

---

| Field | Purpose |
|---|---|
| chat_id | Telegram chat identifier |
| telegram_user_id | Telegram user identifier |
| username | Telegram username |
| lang | User language |
| flow_state | Current workflow stage |
| step | Current intake step |
| last_question | Last question sent |
| voice_concat | Consolidated transcripts |
| voice_count | Number of voice notes |
| ai_summary | AI-generated summary |
| client_summary | Client-oriented summary |
| tech_summary | Technical summary |
| status | Lead status |
| created_at | Record creation timestamp |
| updated_at | Last update timestamp |
| flow3_opened_at | Follow-up window opening time |
| flow3_closed_at | Follow-up window closure time |
| flow3_expires_at | Follow-up expiration time |
